# 📊 Sales Data Analysis Dashboard

## Project Overview
This project analyzes retail sales data to identify key business insights and trends. The goal is to support decision-making using data visualization.

## Tools Used
* Microsoft Excel (Data Cleaning)
* Power BI (Dashboard & Visualization)

## Dataset
The dataset contains information on:
* Order Date
* Region
* Category
* Sub-Category
* Product Name
* Sales
  
## Key Insights
* West region has the highest sales
* Technology is the top-performing category
* Phones are the highest-selling sub-category
* Sales peak in November, indicating seasonal demand
* Top products contribute significantly to total revenue

## Dashboard Features
* KPI card showing total sales
* Sales by Region (Bar Chart)
* Sales by Category (Pie Chart)
* Monthly Sales Trend (Line Chart)
* Top 5 Products (Bar Chart)
* Interactive slicers for Region and Category
